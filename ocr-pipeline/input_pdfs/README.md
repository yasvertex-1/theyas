# Input PDFs Folder

Place your Arabic PDF documents here that you want to extract text from.

## Supported Files:
- PDF files containing scanned Arabic text
- Multi-page documents
- Various image qualities (higher DPI generally produces better results)

## Usage:
1. Copy your Arabic PDF files into this folder
2. Run the extraction pipeline
3. Extracted PDFs will be saved in the `../output_pdfs/` folder

## File Naming:
- Use descriptive names for your PDFs
- Avoid special characters in filenames
- The system will create output files with "_extracted" suffix
